# Wireless Sensor Station (ESP32)

Minimal PlatformIO project for the ESP32-based Wireless Sensor Station used in the Embedded Lab.

## Build & Upload

Prerequisites:
- Install PlatformIO (VS Code extension or CLI)

Build:

```bash
pio run
```

Upload (auto-detects serial port):

```bash
pio run -t upload
```

Open [src/main.cpp](src/main.cpp#L1) to edit the firmware.

Files added by init:
- [.vscode/tasks.json](.vscode/tasks.json)
- [.vscode/launch.json](.vscode/launch.json)
- [.vscode/settings.json](.vscode/settings.json)
